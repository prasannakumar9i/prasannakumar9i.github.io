# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/prasannakumar9i/prasannakumar9i.github.io.git
```

Navigate to the project

```bash
cd prasannakumar9i.github.io
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Build

```bash
npm run build
```
