## 🚀 Core Features

* **Dynamic Project Fetching:** Automatically pulls and displays my latest public repositories directly from the GitHub API, falling back to local data if the API rate limit is reached.
* **Interactive UI:** Built-in 3D tilt hover effects on project cards, smooth scrolling, and scroll-indicator progress bars.
* **Asynchronous Contact Form:** Processed via AJAX/Fetch to handle user messages without refreshing the page, complete with loading spinners and custom success/error toast notifications.
